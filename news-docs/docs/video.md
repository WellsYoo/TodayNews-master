# 视频接口

## 顶部标题接口

- 请求方式：GET

- 请求地址：/video_api/get_category/v1/?

- 请求参数

| 参数              | 类型     | 是否必须 | 描述      | 示例                                       |
| --------------- | ------ | :--: | ------- | ---------------------------------------- |
| device_id       | Int    |  N   | 设备 ID   | 8800803362                               |
| iid             | Int    |  N   | 未知      | 14486549076                              |
