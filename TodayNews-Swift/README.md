# TodayNews-Swift
## Swift开源项目-模仿今日头条-基于 Swift3.0。

## 环境设置

-  项目环境
 -  Xcode 8.3.2
 -  Swift 3.1
 -  iOS 9.3 +
-  使用 cocoaPods 管理第三方库， 如果电脑没有安装 cocoapods，请先安装 cocoapods，cocoapods 版本为 1.2.0，安装方式请参考  [iOS-CocoaPod升级cocoapods1.2.0 (适配swift3)](http://blog.csdn.net/qxuewei/article/details/53113920)。
